## 📌 Overview
This project focuses on analyzing retail sales data to uncover trends, patterns, and actionable business insights.

## 🎯 Objectives
- Understand sales performance across categories and regions
- Identify top-performing products
- Analyze profitability trends
- Provide data-driven recommendations



## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🧹 Data Cleaning
- Removed missing values and duplicates
- Converted date columns into proper format
- Created new features such as Month and Year

## 📊 Analysis Performed
- Sales and profit analysis by category
- Regional performance analysis
- Monthly sales trend analysis
- Product-level performance analysis
- Sales vs Profit relationship

## 🔍 Key Insights
- Technology category generates the highest sales
- Some categories show high revenue but lower profit margins
- Sales performance varies significantly across regions
- A small number of products contribute heavily to total sales

## 💡 Recommendations
- Focus on high-performing categories like Technology
- Improve profitability of low-margin products
- Expand strategies in top-performing regions
- Optimize pricing and cost structures
